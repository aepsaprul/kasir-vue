bantu saya membuat aplikasi kasir dengan spesifikasi:

teknologi:
1. frontend dengan vuejs versi 3.
2. backend api dengan nodejs + express js.
3. database mysql.
4. template dengan CoreUI.

fitur-fitur:
1. otentikasi
2. profile
3. manajemen/master
    - kategori produk
    - satuan/unit
    - produk (bisa mengatur: harga per produk, harga jual/retail, harga grosir dengan minimal pembelian)
    - stok
    - pelanggan
    - supplier
    - diskon
4. transaksi
    - pembelian
    - penjualan
    - kasbon
5. laporan
    - laba rugi
    - cash flow

catatan:
- saya sudah download template CoreUI VueJs dan sudah bisa jalankan "npm run dev".
- semua koding dibuat oleh AI dari mulai buat tabel database, backend API dan frontend, saya tinggal copy paste.
- buat secara bertahap agar lebih detail dan berikan sedikit penjelasan.
